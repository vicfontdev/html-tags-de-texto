Tags de texto HTML

Vamos falar sobre algumas tags de texto

  • Rasurado: usa-se a tag <s> ou <del>. /p>

    Exemplo: Eu não gosto gosto de açaí.

    - Use <s> quando quiser riscar texto sem um contexto de remoção.
    - Use <del> quando o texto foi realmente removido ou excluído, pois isso dá um significado semântico ao conteúdo.

  • Negrito: usa-se a tag <b> ou <strong>

    Exemplo: Palavra em negrito.

    A tag <b> é usada apenas para aplicar o estilo de negrito, sem transmitir nenhum significado semântico enquanto <strong> é usada para indicar importância ou ênfase forte no texto. Também aplica o estilo de negrito, mas com um significado semântico adicional.

  • Itálico: usa-se a tag <i> ou <em>

    Exemplo: Palavra em itálico.

    A tag <i> é usada apenas para aplicar o estilo de itálico, sem qualquer significado semântico enquanto <em> é usada para dar ênfase ao texto, aplicando o estilo de itálico, mas com um significado semântico adicional. É recomendada quando o texto realmente precisa de destaque em leitura.

  • Quebra de linha: usa-se a tag <br>

    Ela é usada para inserir uma quebra de linha em um texto sem iniciar um novo parágrafo.

    Exemplo:
    <p>Este é um texto com uma quebra de linha.<br>Agora, este texto aparece na linha de baixo.</p>

    Isso resultará em:
    Este é um texto com uma quebra de linha.
    Agora, este texto aparece na linha de baixo.

    A tag <br> é uma tag auto-fechada, ou seja, não precisa de uma tag de fechamento como outras (por exemplo, <br/> também é aceito, especialmente em XHTML).

  • Cria uma linha na tela: usa-se a tag <hr>

    Essa tag insere uma linha divisória e é geralmente usada para separar seções de conteúdo.

    Exemplo: <p>Texto antes da linha.</p> <hr> <p>Texto depois da linha.</p>

    Isso resultará em:

    Texto antes da linha.

    Texto depois da linha.

    A <hr> é uma tag auto-fechada, ou seja, não precisa de uma tag de fechamento (similar à <br>), e também pode ser escrita como <hr/> para compatibilidade com XHTML.
